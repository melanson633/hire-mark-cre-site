# Source Context Guidance

## Purpose

Store minimal copied context from upstream workspaces so future agents can route efficiently without depending on or bulk-reading `Job_Search`.

## Routing

- `ai-case-studies/` for copied case-study routing docs and public-safe artifact notes
- `ai-side-hustle/` for offer-development strategy, AI automation consulting concepts, and reusable image/prompt direction copied from the master profile workspace or local approved concepting
- `linkedin-posts/` for post-specific production notes, captions, and source README files

## Working Rules

- Treat these files as reference inputs, not public site copy.
- Do not place raw PII, proprietary workbook evidence, or private job-search notes here.
- Distill publishable claims into `src/content/`.
- Put web-addressable exports in `public/`.
