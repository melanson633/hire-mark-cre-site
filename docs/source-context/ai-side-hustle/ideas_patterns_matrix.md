# Ideas & Patterns Matrix

Living, append-only canvas. Add rows freely. Promote validated rows to `revenue_map.md` once an offer is drafted, target list built, or first paid pilot landed.

## Conventions

- **Pattern ID** — references `capability_patterns.md` (P1–P7). Multiple OK.
- **Vertical** — buyer industry/niche.
- **Offer Shape** — productized / retainer / project / license / audit / cohort.
- **$ Band** — rough price expectation.
- **Edge** — why Mark wins this vs. a generic AI freelancer (1–2 phrases).
- **Status** — `idea` → `drafting` → `outreach` → `pilot` → `live` → `paused` → `killed`.
- **Wiki Signal** — quoted phrase from `wiki_ai-side-business_snapshot.md` if applicable.
- **Notes** — last-touched context, blocker, or next action.
