# Landing Page Image Fit Rubric

Purpose: score the refined image ideas against the current SMB AI automation landing page aesthetic before generating website image candidates.

Current page aesthetic: editorial, quiet, operational, warm off-white, oversized serif type, black CTAs, muted status-color accents, bordered cards, square desktop image frame with mobile wide-crop behavior.

## Minimal Rubric

Score each criterion from `0` to `2`, for a `10` point maximum.

- `Aesthetic fit`: matches the page's refined editorial style without feeling like social-post art.
- `Message fit`: reinforces inboxes, spreadsheets, PDFs, routing, exceptions, reporting, and operational calm.
- `Crop resilience`: works in the current square hero frame and remains coherent in a mobile wide crop.
- `Visual restraint`: has enough negative space and avoids dense dashboard/cartoon clutter.
- `Implementation usefulness`: can plausibly replace the current Nero/Rome image or support a near-term landing section.
